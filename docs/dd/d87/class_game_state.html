Detailed Description

The GameState class is used as a layer to provide Game functions such as Game::loadScene to classes such as FixedUpdatingSystem without exposing sensitive functions like Game::uninitialize or Game::resetVitals.

Essentially: FixedUpdatingSystem, RenderingSystem, and UpdatingSystem should not know the Game class exists.
